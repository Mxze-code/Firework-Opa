# Ascend — Design System

## Color Palette (Red & Black)

| Role | Hex | Usage |
|------|-----|-------|
| **Background Primary** | `#0a0a0a` | Main app background |
| **Background Secondary** | `#111111` | Nav, elevated surfaces |
| **Background Tertiary** | `#1a1a1a` | Hover states |
| **Background Elevated** | `#222222` | Inputs, progress tracks |
| **Background Card** | `#141414` | Cards, panels |
| **Text Primary** | `#f5f5f5` | Headings, primary content |
| **Text Secondary** | `#a3a3a3` | Secondary content |
| **Text Muted** | `#737373` | Labels, hints |
| **Accent Primary** | `#dc2626` | CTAs, highlights |
| **Accent Secondary** | `#b91c1c` | Hover on accent |
| **XP Accent** | `#ef4444` | XP, progress |
| **Success** | `#22c55e` | Completed states |

## Typography

**Space Grotesk** — Geometric, modern.

## Copy Guidelines

- **Serious, premium, performance-oriented**
- Use: Daily targets, Progress, Consistency, Level, Performance
- Avoid: Fantasy RPG language, fake achievements, exaggerated claims

## Empty States

- Motivating prompts
- Clear CTAs (Log workout, Set targets, etc.)
- No fake or placeholder data
